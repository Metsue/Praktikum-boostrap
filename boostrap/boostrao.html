Materi Bootstrap

Apa itu Bootstrap?

Bootstrap adalah framework open-source berbasis HTML, CSS, dan JavaScript untuk membuat website yang cepat, responsif, dan mobile-first.

Bootstrap CSS

Bootstrap CSS menyediakan class siap pakai untuk grid, layout, tombol, form, alert, card, dan lainnya.

Bootstrap JS

Bootstrap JS mengaktifkan komponen interaktif seperti modal, carousel, dropdown, tooltip, dan collapse. Tidak lagi membutuhkan jQuery.

Keunggulan

Responsif secara default, mudah digunakan, dokumentasi lengkap, dan komunitas besar yang aktif.
